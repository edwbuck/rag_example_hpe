# Cat facts

Cat facts are downloaded from 
[cat-facts](https://huggingface.co/ngxson/demo_simple_rag_py/resolve/main/cat-facts.txt) 
and are an extension of possibly interesting tidbits including references from:

- [www.factretriever.com](https://www.factretriever.com/cat-facts)
- other unknown sources.

The file format is one fact per line, wrtten as an English sentence, terminated
by a full stop.

## Data cleansing

The database of cat facts has not had any additional data cleansing applied, it
is an example database that is assumed to be clean for the purposes of
generating a RAG vector data source.
